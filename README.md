# Kata1
Para esta práctica voy a usar IntelliJ junto con Git. El reto es gestionar todo el proyecto (crear clases, moverse por los menús, etc.) usando únicamente atajos de teclado para ser más eficiente.
Además, voy a llevar un control de versiones estricto, haciendo commits a menudo y detallando bien qué hago en cada uno para que el historial quede limpio.
